Exosite

Este tutorial irá explicar como configurar um Portal gratuitos, criar uma conta,dispositivo e dados associado, enviar dados para esse dispositivo, e exibir os dados em um widget na sua página inicial do painel.

Pré-Requisitos

  • Computador com acesso a internet

  • Conta de Email Ativa

Criando Conta

Para começar, você precisa criar uma conta. Se você já possui uma conta, você pode avançar para a "criar um novo dispositivo" deste documento.
  1. Acessar o site do Exosite’s Account Sign Up web page.
  2. Complete o formulário e criar a conta.
  3. Você receberá um e-mail do portals.exosite.com que lhe pede para ativar sua conta. Clique no link apropriado no e-mail para ativar sua conta.
  4. Quando concluída, faça o login na sua conta Portals login page usando seu endereço de e-mail ea senha que você acabou de criar..
    5. Portal

    Exemplo

    Criando novo Device

    Para os fins deste tutorial, este dispositivo será temperatura para o qual você pode gravar dados da temperatura Ambiente.

    1. Selecione Devices na barra lateral e na pagina, click "+ Add Device." A Device Setup *window will open. On the *Setup Type tab, selecione a opção "I want to create a generic device." Click Continue.
    2. Device Setup :
      1. Verifique se o tipo de dispositivo menu drop-down é definido como genérico..
      2. Selecione seu fuso horário.
      3. Digite um local no campo Device Location A localização pode ser qualquer coisa que indica a localização do dispositivo, mas não pode ser deixado vazio. No exemplo a seguir, a localização é definida para João Pessoa.
      4. Click Continue.
      5. Em seguida especifique um nome para o dispositivo.

        6. Agora você deve ver o dispositivo que você acabou de criar. Para visualizar as informações pertinentes sobre o dispositivo, clique no dispositivo para abrir a janela de Informações do dispositivo.

    Criando novo Dado

    Para gravar dados no dispositivo que você acabou de criar, você precisa primeiro criar uma porta de dados para conter os dados. A porta de dados contém uma série de pontos com registro de tempo e pode viver dentro de um dispositivo ou de um portal.

    1. Selecione "Data" do menu Portal para exibir uma lista de todos os dispositivos em seu portal e suas portas de dados associados.
    2. Em Data, click "+ Add Data."
    3. Abrirar a janela da Data Setup:
    4. Selecione o rótulo do botão de rádio "From a Device" já que você está adicionando uma porta de dados para o dispositivo que você acabou de criar.
    5. Selecione o dispositivo que você acabou de criar
    6. Click Continue.

      Configurando o dado

    1. Em *Data Source Name *field. Escolha o nome fantasia para descrever o seu dado.
    2. Selecione o tipo do dados em Data Source Format. No caso de temperatura podemos escolher Float
    3. Escolha a Unidade no Dado. No exemplo aqui ºC/li>
    4. Este é o nome de dispositivos que será usado para identificar este dado ao acessar interfaces (APIs).
    5. Click Submit.


Personalizando Widgets

Agora que você tem um dado configurado à porta de dados, você pode criar um widget para capturar essa informação na sua página inicial do painel.

  1. Selecione "Home" no menu para retornar a pagina principal.
  2. Click "Add Widget" para adicionar um novo Widget.
  3. Selecione Gauge em Widget Type na opção.
  4. Block Title é o Título do Widget.
  5. Click Continue.

  6. Na guia de Configuração, você pode configurar muitas características do widget.


  7. Click Submit.

  8. Deverá da uma mensagem Widget added successfully. Click Quit para retornar para seu dashboard.

Por padrão, o novo widget ocupa o primeiro espaço vazio no painel.

Observe que os relatórios de widget recarrega com base no tempo de atualização que você definiu. Você pode usar cURL ou EXOline para gravar dados eo widget irá atualizar automaticamente o número que aparece quando ele recarrega.

HTTP API Overview

This is a lightweight HTTP-based API for writing to and reading from the Exosite One Platform. It is intended for bandwidth-constrained applications. It provides the ability to write new data points or read the latest data point.

Data Procedures
